## How to run it

For you to run this project on your computer, follow this steps:

1. Clone repository
```
$ git clone https://github.com/MelissaPerez09/Proyecto01-CC3062
```

2. Install all dependencies
```
$ yarn add vite
```

3. Running on localhost
```
$ yarn dev
```
